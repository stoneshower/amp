amp-ad

  • ライブラリ - 要 (不要だったが推奨)

amp-adは、AMPのページ上で広告を表示できる専用タグです。

Position Fixed(amp lightboxを除く)では使用できない。

HTTPS対応のAdのみ表示できる

placeholder

fallback

Running ads from a custom domain

Ad Network List => https://www.ampproject.org/docs/reference/components/amp-ad

Ads are loaded using a the tag given the type of the ad network and name value pairs of configuration. This is an example for the A9 network:

<amp-ad
    width="300"
    height="250"
    layout="responsive"
    type="a9"
    data-aax_size="300x250"
    data-aax_pubname="test123"
    data-aax_src="302">
    <div placeholder>ad placeholder!!!</div>
    <div fallback>ad fallback!!!</div>
</amp-ad>
